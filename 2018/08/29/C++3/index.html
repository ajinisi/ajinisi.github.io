C++常见问题之声明定义和初始化

变量

const变量

因为常量在定义后就不能被修改，所以定义时必须初始化

数组

一个数组不能用另外一个数组初始化，也不能将一个数组赋值给另一个数组

指针

有效的指针必然是以下三种状态之一：保存一个特定对象的地址；指向某个对象后面的另一对象；或者是 0 值。所以对指针进行初始化或赋值只能使用以下四种类型的值

  • 值常量表达式
  • 类型匹配的对象的地址
  • 另一对象末的下一地址
  • 同类型的另一个有效指针
