Git的使用之多人合作二

首先要明白github上多人合作主要有三种模式，第一种就是把你的伙伴的公钥加到你项目的公钥；第二种建一个组织organization，然后在该组织下建一个team 把你的伙伴加到team 然后把项目放到team下；第三种就是你的伙伴fork你的项目，然后给你发 pull request，由你来决定是否 merge，这是大型开源项目的用法。这里我们采用第三种。
